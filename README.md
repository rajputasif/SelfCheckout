# Self checkout with CV

## Pre-requisites
- Python >= 3.12
- virtualenv
- git
- GPU (for faster inference)

## Installation
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Usage
Modify the `settings.yaml` file to suit your needs.
```
(.venv) $ python checkout.py
```
